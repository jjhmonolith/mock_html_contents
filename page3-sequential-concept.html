어떤 것들은 나열 순서 자체가 중요한 의미를 가지고, 어떤 것들은 순서가 상대적으로 덜 중요해요.

🍳

요리 레시피

계란후라이 만들기: 팬 달구기 → 기름 두르기 → 계란 깨기

왜 순서가 중요할까요?
순서를 바꾸면 완전히 다른 결과가 나와요! 기름 없이 계란을 먼저 넣으면 팬에 달라붙고, 팬을 안 달구고 요리하면 제대로 익지 않아요. 😅
+
📅

학교 시간표

1교시 국어 → 2교시 수학 → 3교시 체육 → 4교시 과학

왜 순서가 중요할까요?
체육 다음에 수학이 있으면 땀 흘린 상태로 수학 공부를 해야 해요. 과목 순서에 따라 하루 컨디션이 달라지죠! 📚
+
🚌

버스 정류장

학교앞 → 시청 → 병원 → 마트 → 우리집

왜 순서가 중요할까요?
정류장 순서가 바뀌면 내릴 곳을 놓치거나 엉뚱한 곳에 가게 돼요. 버스 노선의 순서는 우리가 목적지에 도착하는 방법이에요! 🎯
💚 순서가 덜 중요한 것들
순서가 바뀌어도 큰 문제없어요!
+
✏️

연필꽂이

볼펜, 연필, 지우개, 자, 형광펜이 들어있어요

왜 순서가 덜 중요할까요?
어떤 순서로 넣어도 문구용품으로서의 기능은 똑같아요. 필요할 때 꺼내 쓰면 되거든요! (단, 무지개 색깔 순서로 예쁘게 꽂고 싶다면 순서가 중요해질 수도 있어요 🌈)
+
🍿

과자 봉지

봉지 안에 여러 개의 과자가 들어있어요

왜 순서가 덜 중요할까요?
어떤 과자를 먼저 먹든 맛있기는 마찬가지예요! 과자들이 봉지 안에서 어떤 순서로 있든 상관없죠. 😋

같은 '모음'이라도 어떤 것들은 나열 순서 자체가 의미예요. 순서가 중요한 데이터를 우리는 '순차적 데이터'라고 부릅니다! 순서가 바뀌면 전혀 다른 의미가 되는 거죠. 🎯

🎉 맞아요! 순서를 말할 때 꼭 필요한 게 인덱스/위치입니다!
